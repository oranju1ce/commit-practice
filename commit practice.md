Git commands:
1. Git init - это «точка старта». Она превращает обычную папку на твоём компьютере в локальный Git-репозиторий.
Что происходит внутри?
Когда ты вводишь эту команду, Git создает в папке скрытую директорию .git. Именно там хранится вся история изменений, настройки и служебная информация. Если ты удалишь эту папку, проект перестанет быть репозиторием, и вся история коммитов исчезнет (хотя сами файлы в папке останутся).
